# supabase-swr

A component library to use [supabase-js](https://github.com/supabase/supabase-js) with [swr](https://github.com/vercel/swr)

## Install

Using npm.

```shell
npm install supabase-swr
```

Using yarn.

```shell
yarn add supabase-swr
```
